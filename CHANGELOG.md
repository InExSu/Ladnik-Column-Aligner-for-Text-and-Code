# Change Log

All notable changes to the project will be documented in this file.

## [1.0.0] - 2026-02-15

### Added
- Main text alignment functionality based on delimiters
- Support for multiple delimiters with length-based priority
- Configuration via .ladnikrc.json
- Support for language-specific rules
- Commands: ladnik.alignSelection, ladnik.alignFile, ladnik.configure
- Keyboard shortcut: Ctrl+Alt+L / Cmd+Alt+L
- Context menu in editor
- Unit tests for all main components
- Integration tests
- Unicode and emoji support
- Line ignoring by prefixes (comments, etc.)
- Configuration validation
- Documentation and usage examples

### Changed
- None

### Fixed
- None

## [0.1.0] - 2026-02-15

### Added
- Initial release of Ladnik Column Aligner
- Basic alignment logic
- Support for main delimiters (=, :, =>, ::)
- VS Code settings configuration
